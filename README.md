# kargo

Kono's personal cargo++.
Wraps over cargo commands to generate personal config and boilerplate.

## Usage

```
kargo new foo 
kargo wrk mylib mybin
kargo rld
```

## Install

### From crates.io

```
cargo install kargo
```

### From source

```
git clone https://github.com/Kono-o/kargo
cd kargo
cargo install --path .
```
